- [SendGrid](https://github.com/stephenb/sendgrid) gem for email delivery.
- Continuous testing with [Guard](https://github.com/guard/guard).
- Thin server for lightweight performance.
- MySQL database.
- ERB template engine.

## Usage

To start using this template, follow these standard Rails setup steps:

1. Install **Ruby** version 2.2.3 (via [RVM](https://github.com/rvm/rvm), [rbenv](https://github.com/sstephenson/rbenv), or your preferred method).

2. Clone the repository and install dependencies:

```sh
git clone git@github.com:dreamingechoes/rails-blank-template-app.git YOUR_APP_NAME_HERE
cd YOUR_APP_NAME_HERE
bundle install
rake db:setup
rake db:seed
```

3. Start the server and check everything is working:

```sh
rails s
```
